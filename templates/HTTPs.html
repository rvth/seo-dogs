What is HTTPs?

HTTPS stands for HyperText Transfer Protocol Secure and is the main protocol used by websites to transfer data securely across the internet.

As a secure protocol, HTTPs protects the communication between a user's browser and a website's server from being intercepted. This means that sites can are safe for users to provide private details.

The difference between HTTP and HTTPs

HTTPS essentially encrypts and secures a HTTP request to ensure that the connection cannot be intercepted, and to keep user's information safe.

The impact of HTTPs on users

Using HTTPS protocols on a site provides a positive impact on users. This is because it helps to keep their information safe from potential attackers. It also helps users to trust the website by informing them that it is safe to browse the website and, in the case of an eCommerce site, to purchase from the site.

The impact of HTTPs on SEO

There is also a positive impact to SEO by using the HTTPS protocol, this is largely because search engines such as Google use it as a ranking factor. Having a secure website helps search engines understand that a website is safe for them to display to search queries.

If a site does not yet contain the HTTPS protocol, a majority of browsers will display a warning message when users try to visit the site. This warning message may also be seen by search engine crawlers and could lead to them dropping a page from the index.

Search engines may also display a warning in Search Results Listing pages, which may prevent users from visiting a website altogether.
