Crawling

Search engine bots will crawl a website in order to discover pages and understand the content.

Rendering

Indexing

If a page meets certain dependincies it can be organised and added to the search engine index.

🐾 Explore

Robots.txt

Robots.txt is be used to inform search engines of pages you would not like them to crawl.

🐾 Explore

Noindex

This small tag can be added to the HTML of a page to inform search engines not to include the page in the index.

🐾 Explore

Nofollow

The nofollow tag is used to tell search engine crawlers not to follow a link, or pass equity between pages.

🐾 Explore

X-robots

Similar to a noindex tag, X-robots is used to control the indexing of a page and is placed in the HTTP response header.

🐾 Explore

Canonicals

Used for managing duplicate content, canonical tags are a signal to inform search engines of your preferred page for indexing.
