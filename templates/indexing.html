What is search engine indexing and how does it work?

Indexing is the process of gathering all of the content found during the crawling process. Once a page has been found and rendered, the search engine crawlers will analyse the content and then store and organise everything found within its index.

Having all content stored within an index makes it super fast for search engines to retrieve information once a user searches for a particular query. Searching through every individual page on the web once a query has been made would be an incredibly slow process, so the index helps them to organise and deliver content relevant to a user's query.

You can control which pages are added to the index by using some meta directives such as noindex and canonical tags. You can find out more about noindex tags here and canonical tags here.
